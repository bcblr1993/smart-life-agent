# 🦞 小红书AI运营助手 - 龙虾

> 中关村北纬龙虾大赛参赛作品

## 📕 项目简介

这是一个基于 AI Agent 的小红书运营助手，能够：
- 🤖 自动生成高质量种草文案
- 🔥 分析热门话题趋势
- 🎨 生成 AI 绘画提示词
- 💬 智能回复评论

## 🏆 参赛赛道

**生产力龙虾** - 解决实际工作场景需求

## 🚀 功能演示

### 1. 生成种草文案
```bash
python3 xiaohongshu_agent.py generate_post "降噪耳机" "数码"
```

### 2. 热点分析
```bash
python3 xiaohongshu_agent.py analyze_hot "美妆"
```

### 3. AI 图片提示词
```bash
python3 xiaohongshu_agent.py generate_image_prompt "护肤品" "小清新"
```

### 4. 评论回复
```bash
python3 xiaohongshu_agent.py generate_reply "好评"
```

### 5. 完整演示
```bash
bash demo.sh
```

| 优势 | 说明 |
