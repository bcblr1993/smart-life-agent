# 🦞 Smart Life Agent - 智能生活助手

> 中关村北纬龙虾大赛参赛作品 | 赛道：生活龙虾

## 📖 项目简介

Smart Life Agent 是一款基于 AI Agent 的智能生活助手，帮助用户高效管理日常生活，提升生活品质。

## 🎯 核心功能

### 1. 📅 智能日程管理
- 自动识别和整理日程
- 智能提醒，不再错过重要事项
- 日程冲突检测和优化

### 2. 🍳 食谱推荐
- 根据用户口味和食材推荐菜谱
- 营养均衡分析
- 购物清单自动生成

### 3. ✈️ 旅行规划
- 智能行程规划
- 景点推荐和避坑指南
- 预算控制和费用分析

### 4. 💰 生活记账
- 语音快速记账
- 消费分析报告
- 预算提醒

### 5. 📚 学习助手
- 学习计划制定
- 知识点总结
- 复习提醒

## 🛠 快速开始

### 本地运行

```bash
# 克隆项目
git clone https://github.com/bcblr1993/smart-life-agent.git
cd smart-life-agent

# 安装依赖
pip install -r requirements.txt

# 运行演示
python demo.py
```

### Docker 运行

```bash
docker build -t smart-life-agent .
docker run -p 8080:8080 smart-life-agent
```
