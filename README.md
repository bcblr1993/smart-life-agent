# 🦞 Smart Life Agent - 智能育儿生活助手

> 中关村北纬龙虾大赛参赛作品 | 赛道：生活龙虾

## 📖 项目简介

Smart Life Agent 是一款专注于**育儿生活**的 AI 智能助手，帮助父母科学育儿、记录成长、规划学习。

## 🎯 核心功能（四大模块）

### 1. 🍳 儿童食谱推荐
- 根据年龄段推荐营养餐
- 区分早餐/午餐/晚餐
- 简单易做的亲子料理
- 营养分析和建议

### 2. 📖 儿童睡前故事
- 原创温馨睡前故事
- 适合不同年龄段
- 寓意深刻，培养品格
- 每日更新

### 3. 📝 儿童日常记录
- 生长发育档案（身高/体重）
- 饮食记录
- 情绪追踪
- 成长相册整理

### 4. ⏰ 学习提醒与规划
- 幼小衔接学习计划
- 习惯养成追踪
- 定时提醒功能
- 学习进度记录

## 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/bcblr1993/smart-life-agent.git
cd smart-life-agent

# 安装依赖
pip install -r requirements.txt

# 运行演示
python3 demo.py

# 对话测试
python3 demo.py 今天吃什么
python3 demo.py 讲个睡前故事
python3 demo.py 记录今天的表现
python3 demo.py 今天的学习计划
```
